# TomShelBot
Telegram bot made for local community, for fun.

## Commands
```
Итак.. вот список того, что я умею.. (27): 

/all (алиасы: /atall, /все, /сылште)
- Упомянуть всех, кого бот знает.

/beat (алиасы: /побить, /knock, /fuck, /tear)
- Изрезать... КОЗЫРЬКОМ

/check (алиасы: /lookup, /who, /чек, /кто)
- Основная информация из сведений семьи Шелби.

/clapmute (алиасы: /кляп, /muteclap)
- Вставить кляп. Именно кляп! Особый вид мута

/coin (алиасы: /throw, /flip)
- Подбросить монетку

/f
- F in chat

/go (алиасы: /lets, /погнали)
- Отправит вопрос со слов 'Погнали ...'.

/goose (алиасы: /гусь, /кря, /quack)
- Закинуть гуся.

/id (алиасы: /chatid)
- Узнать id чата

/info (алиасы: /start)
- Основаня информация о боте

/mute (алиасы: /мут, /молчи, /shutup, /заткнись)
- Под угрозой расстрела сообещния запретить человеку писать.

/pcheck
- Проверить наказания у гражданина

/random (алиасы: /number, /randomit)
- Получить рандомное число (диапазон тоже можно указать).

/say
- Сказать что-то от имени бота

/sticksoff (алиасы: /offstickers, /banstickers, /банстик, /banstick)
- Запрещать извергам слать стикеры.

/summon
- Отправить призыв прописать комманду

/test
- Тестовая комманда.

/thelp (алиасы: /помощь)
- Помощь по коммандам бота

/toxic
- Поставить алерт на токсика

/unmute (алиасы: /muteoff, /offmute, /pardon)
- Размутить. Выдать.. Право.. Голоса

/unsummon
- Сбросить призыв прописать комманду

/wall (алиасы: /стенка, /стена, /stalemate)
- Построить непробиваемую стенку.
```
